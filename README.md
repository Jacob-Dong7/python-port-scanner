# Python Port Scanner

A beginner cybersecurity project that checks whether a TCP port is open or closed on a target device.

</div>

---

## Description

This project uses Python's built-in socket module to create a TCP connection attempt to a given IP address or domain name and port number.

If the connection succeeds, the port is marked as open 
If the connection fails, the port is marked as closed

---

## Features

<ul>
  <li>Accepts a target IP address or domain name</li>
  <li>Accepts a port number</li>
  <li>Creates an IPv4 TCP socket</li>
  <li>Sets a timeout to avoid long waits</li>
  <li>Checks whether the port is open or closed</li>
  <li>Prints the result clearly</li>
</ul>
